# Predicción del Comportamiento de Compra de Clientes

## Descripción

Proyecto de Machine Learning desarrollado con Python y Scikit-Learn para predecir la respuesta de clientes a campañas de marketing.

## Objetivos

- Analizar datos de clientes.
- Realizar limpieza y transformación de datos.
- Entrenar un modelo de Regresión Logística.
- Evaluar el rendimiento del modelo.

## Tecnologías

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Resultados

El modelo fue evaluado mediante Accuracy, Precision, Recall y F1-Score para medir su capacidad predictiva.
